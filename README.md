<!-- ---------------- -->
Project Overview
<!-- ---------------- -->

This is a full-stack TeleHealth Appointment Management System that allows users to:

View their appointments

Schedule new appointments with doctors

View prescriptions

Preview a video consultation interface

The application is built with a modern MERN-style stack, focusing on clean UI, proper validation, and scalable backend APIs.\

Features

<!-- ------------------- -->
Appointments listing
<!-- -------------------- -->

Schedule appointment with validation (future date & time)

Doctor selection

Prescription listing

Video call preview screen

Protected routes with login

Responsive admin dashboard UI

<!-- ----------------------------- -->
Technologies & Libraries Used
<!-- ----------------------------- -->

Frontend

React (Vite)

React Router DOM

Tailwind CSS

Axios

Flatpickr (date & time picker)

Custom UI components (Select, Layout, Sidebar)

Backend

Node.js

NestJS

MongoDB (MongoDB Atlas)

Mongoose

Hosting

Frontend: Vercel / Netlify

Backend: Render

Database: MongoDB Atlas
